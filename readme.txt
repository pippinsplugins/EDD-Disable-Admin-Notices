A simple add-on plugin for Easy Digital Downloads that disables the admin sale notices

== Description ==

This plugin is an extension for [Easy Digital Downloads](https://easydigitaldownloads.com) and simply disables the sale notice emails that get sent to admin users whenever a sale is processed through Easy Digital Downloads.

If you do not like receiving emails each time a new sale is made, install this simple extension to get rid of them.
